# Python Mini Game

## Тетрис (PyGame)

Простой Тетрис с управлением стрелками на клавиатуре.

### Установка

```bash
pip install -r requirements.txt
```

### Запуск

```bash
python Game_Tetris/main.py
```

### Управление

- Стрелка влево / Стрелка вправо: перемещение фигуры
- Стрелка вверх: поворот фигуры
- Стрелка вниз: мягкое падение
- R: перезапуск после конца игры

## Galaxy (PyGame)

Аркадный шутер в духе Galaxians: ваш корабль находится внизу и уничтожает атакующие волны врагов.
Вся графика нарисована примитивами PyGame (без внешних графических ассетов).

### Запуск

```bash
python Game_Galaxy/main.py
```

### Управление

- Стрелка влево / Стрелка вправо: перемещение корабля
- Пробел: выстрел
- R: перезапуск после конца игры

## Memory (PyGame)

Игра на запоминание карточек с уровнями:
начинаете с 4 карточек и на каждом новом уровне добавляется +2 карточки.
Открывайте две карточки, находите пары, убирайте их с поля и продолжайте бесконечно без состояния проигрыша.
Символы на карточках — числа и цветные геометрические фигуры.

### Запуск

```bash
python Game_Memory/main.py
```

### Управление

- Левая кнопка мыши: перевернуть карточку

## Snake (PyGame)

Классическая Snake на сетке с управлением стрелками.
Ешьте еду, чтобы расти, и избегайте столкновений со стенами и собственным телом.

### Запуск

```bash
python Game_Snake/main.py
```

### Управление

- Стрелка влево / Стрелка вправо / Стрелка вверх / Стрелка вниз: движение змейки
- R: перезапуск после конца игры

## Симулятор парковки (PyGame)

Вид сверху, симулятор парковки с полуреалистичным управлением автомобилем.
Двигайтесь вперед и назад, аккуратно объезжайте препятствия и полностью паркуйтесь в гараже в форме буквы P, не касаясь стен.
Если текущая раскладка неудобна, можно сгенерировать новый случайный уровень.

### Запуск

```bash
python Game_Parking/main.py
```

### Управление

- Стрелка вверх: ускорение вперед
- Стрелка вниз: движение назад
- Стрелка влево / Стрелка вправо: поворот передних колес
- R: перезапуск после победы или аварии
- N: сгенерировать новый случайный уровень
