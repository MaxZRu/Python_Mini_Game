# Arcanoid на pygame

Мини-игра в стиле Арканоид: управляйте платформой, отбивайте мяч и разбивайте блоки.

## Возможности

- Платформа, мяч и разрушение блоков.
- Несколько уровней с разной раскладкой и прочностью блоков.
- Счет, жизни и экраны старта/победы/поражения.
- Постепенное ускорение мяча по мере попаданий по блокам.

## Управление

- `A` / `D` или стрелки `Left` / `Right` — движение платформы.
- `Space` — старт игры.
- `R` — перезапуск игры.

## Запуск

1. Перейдите в папку проекта:
   - `cd Game_Arcanoid`
2. Установите зависимости:
   - `pip install -r requirements.txt`
3. Запустите игру:
   - `python main.py`
